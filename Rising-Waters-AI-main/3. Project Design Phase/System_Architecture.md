# System Architecture

## Overview

The Rising Waters system follows a simple Machine Learning web application architecture.

```
           User
             │
             ▼
      HTML Web Interface
             │
             ▼
       Flask Application
             │
             ▼
     XGBoost ML Model
             │
             ▼
   Flood Risk Prediction
             │
             ▼
 AI Safety Recommendations
```

## Components

- User Interface (HTML/CSS)
- Flask Backend
- XGBoost Machine Learning Model
- Prediction Engine
- Result Display
