# Abstract

Floods are among the most destructive natural disasters, affecting millions of people worldwide every year. Early flood prediction plays a significant role in reducing the impact of such disasters.

The Rising Waters project is an AI-powered Flood Prediction System developed using Machine Learning. The system predicts flood probability using environmental parameters such as rainfall, drainage, climate conditions, urbanization, and infrastructure-related factors.

Multiple machine learning algorithms, including Decision Tree, Random Forest, K-Nearest Neighbours (KNN), and XGBoost, were trained and evaluated. Based on the evaluation results, XGBoost was selected as the final prediction model.

The trained model was integrated into a Flask web application that allows users to enter environmental parameters and receive instant flood risk predictions along with AI-generated safety recommendations.

The application is deployed online using Render and maintained using GitHub, making it accessible from anywhere.
